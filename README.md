<h1>Jak uruchomić projekt lokalnie?</h1>
<p>(Linux) Uzyskać prawa do uruchomienia Run.sh</p>
<p>Na przykład: </p>

```javascript
chmod 777 Run.sh
```

<p>Uruchomić skrypt, z poziomu katalogu, w którym się znajduje</p>
<p>Linux:</p>

```javascript
sudo ./Run.sh
```

<p>Windows:</p>

```javascript
.\Run.cmd
```
